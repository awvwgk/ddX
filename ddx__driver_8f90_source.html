57! solvation problem for a given solute. A state must be used with a
58! model (ddx_data). Different states can be used at the same time with

71! STEP 3: Building the required RHSs and electrostatic properties.
72!
73! dd models require two different kind of RHS. The primal RHS which is
74! based on electrostatic properties, and the adjoint RHS which is a
75! representation of the density in spherical harmonics (with extra steps).
76! Furthermore, when computing the forces, electrostatic properties of
77! order +1 are required to compute the force contribution from the RHS.
78!
79! ddLPB is slightly more complicated as its primal RHS requires also
80! the electric field, and if forces are to be computed, requires the
81! electric field gradient.
82!
83! For convenience all the required electrostatic properties are computed
84! together in STEP 3a. The adjoint RHS is computed in STEP 3b.
85
86! STEP 3a: electrostatic properties for primal RHS and for the forces.
87
88start_time = omp_get_wtime()
89
90! The ddx_multipolar_solute provides routines to compute the
91! electrostatic properties up to field gradient for multipolar
92! distributions of arbitrary order, provided that they are given in
93! real spherical harmonics. Here we have charges, so we need to convert
94! them to monopoles. For charges the conversion is simply a scaling by
95! 1/sqrt(four*pi). For more complicated cases, up to the 16-poles, check
96! https://en.wikipedia.org/wiki/Table_of_spherical_harmonics#Real_spherical_harmonics
97allocate(multipoles(1, ddx_data % params % nsph), stat=info)
98if (info .ne. 0) then
99 write(6, *) "Allocation failed in ddx_driver"
100 stop 1
101end if
102multipoles(1, :) = charges/sqrt4pi

104! Get the required electrostatic properties.

107 do_phi = .true.
108 do_e = .true.
109 do_g = .true.
110 else
111 do_phi = .true.
112 do_e = .true.
113 do_g = .false.
114 end if
115else
116 if (ddx_data % params % force .eq. 1) then
117 do_phi = .true.
118 do_e = .true.
119 do_g = .false.

121 do_phi = .true.
122 do_e = .false.
123 do_g = .false.
